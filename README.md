# Student Grade Management System

## Overview
The Student Grade Management System is a robust desktop application designed for educators to efficiently manage student records. Developed as the final project for the COMP1002 - Advanced Python course, this system eliminates the risk of manual calculation errors by automating weighted average/GPA calculations. It features a user-friendly Graphical User Interface (GUI) and ensures data persistence by securely storing student information and grades in structured local files (JSON/CSV). 

**Developer:** Şamil Keklikoğlu 

## Features
* **Student Registration:** Easily add new student profiles to the system.
* **Grade Entry:** Course-based grade input and tracking.
* **Automated Calculations:** Automatic calculation of weighted averages and GPAs.
* **Search Functionality:** Find existing student records quickly using their ID.
* **Data Persistence:** All records are preserved locally (JSON/CSV) between sessions.
* **Visual Interface:** Clean and intuitive desktop UI built with Tkinter.
